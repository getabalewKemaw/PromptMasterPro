===========================================
STEP 1: SETUP DATABASE (Choose One)
===========================================

OPTION A: Free Cloud Database (5 minutes)
------------------------------------------
1. Visit: https://neon.tech
2. Sign up (free)
3. Create new project
4. Copy connection string
5. Paste in backend/.env as DATABASE_URL
6. Done!

OPTION B: Local PostgreSQL (10 minutes)
----------------------------------------
1. Install PostgreSQL
2. Open terminal:
   psql -U postgres
   CREATE DATABASE promptmasterpro;
   \q
3. Update backend/.env:
   DATABASE_URL="postgresql://postgres:YOUR_PASSWORD@localhost:5432/promptmasterpro?schema=public"
4. Done!

===========================================
STEP 2: GET OPENAI API KEY (5 minutes)
===========================================

1. Visit: https://platform.openai.com
2. Sign up / Login
3. Go to API Keys section
4. Create new secret key
5. Copy the key (starts with sk-)
6. Paste in backend/.env as OPENAI_API_KEY
7. Done!

===========================================
STEP 3: CONFIGURE ENVIRONMENT (2 minutes)
===========================================

1. Open: backend/.env
2. Fill in these values:

   DATABASE_URL=<from step 1>
   OPENAI_API_KEY=<from step 2>
   JWT_SECRET=<any random long string>
   PORT=3000
   NODE_ENV=development
   CORS_ORIGIN=http://localhost:8081

3. Save file
4. Done!

===========================================
TROUBLESHOOTING
===========================================

Problem: "Can't reach database"
Solution: Check DATABASE_URL in .env

Problem: "Invalid API key"
Solution: Check OPENAI_API_KEY in .env

Problem: "Port already in use"
Solution: Change PORT in .env to 3001

Problem: "Module not found"
Solution: Run "npm install" in backend folder
